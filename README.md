## Lab Questions

### In plain English describe the function and use of a multiplexer.
A multiplexer takes several inputs and uses a signal to select a single output.
### In plain English describe the function and use of a demultiplexer.
A demultiplexer is the opposite and takes a single input and routes it to one of many outputs based on a signal.
### What other uses might these circuits have? (Think Shannon’s)
In terms of Shannon's, a multiplexer can be used to create any Boolean logic function. What this means is they can also build arithmetic circuits and more.
